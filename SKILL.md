# Vibe Coding Cleanup: Type Fixer

This skill specializes in strictly typing TypeScript codebases.

## Capabilities
1. Identifies `any` and `unknown` types in the codebase.
2. Analyzes the usage context to infer the correct interface or type.
3. Generates and applies the proper TypeScript interfaces.

## Usage
Invoke this skill when the user asks to "fix types", "remove anys", or "improve type safety".
